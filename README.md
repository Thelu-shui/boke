# markdown语法

## 标题标记
# 表示最大的标题（h1)
## 表示2级的标题（h2)
### 表示3级的标题（h3)
#### 表示4级的标题（h4)
##### 表示5级的标题（h5)
###### 表示6级的标题（h6)

##列表
###无序列表
- 用'-'表示无序列表
* 用'*'表示无序列表
+ 用'+'表示无序列表

## 有序列表
1. 用数字+.'1'表示有序列表
